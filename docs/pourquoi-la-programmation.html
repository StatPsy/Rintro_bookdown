Chapter 2 Pourquoi la programmation ?

La programmation est le processus qui consiste à donner des instructions à un ordinateur afin qu’il puisse effectuer des tâches. Ces instructions sont écrites dans un langage que l’ordinateur peut comprendre. Le langage R, utilisé dans ce cours, est un langage de programmation couramment utilisé pour les statistiques et l’analyse de données.

Pour les statistiques et la psychologie, programmer en R offre de nombreuses possibilités, bien plus que les logiciels classiques d’analyse de données : - Automatiser les tâches répétitives d’analyse de données en recherche psychologique - Analyser et visualiser efficacement les données issues d’expériences et d’enquêtes - Reproduire et partager les analyses pour assurer la transparence dans les études psychologiques - Appliquer des méthodes statistiques pour répondre à des questions de recherche en psychologie

Vous pouvez considérer R comme une calculatrice très avancée qui vous permet, avec des connaissances, de réaliser n’importe quelle opération sur vos données et de créer n’importe quelle visualisation pour accompagner vos résultats.

2.1 Exemples de programmation en R

Voici une commande R simple qui additionne deux nombres :

2 + 3

Un des concepts de base est celui des variables. Les variables stockent des informations pour une utilisation ultérieure et peuvent être modifiées au cours du code, par exemple :

a <- 2 b <- 3 a + b

Si on modifie une variable, cela change le résultat :

b <- 4 a + b

Cela peut être très utile si, par exemple, vous souhaitez mettre à jour la variable. Par exemple, simulons un individu qui vieillit au fil des années :

age <- 0 for (annee in 1:18) { age <- age + 1 print(age) if (age == 18) { print(“ça y est je suis majeur”) } }

2.2 Concepts de base

  • Variables : Stockent des informations (comme des nombres ou du texte).
  • Fonctions : Effectuent des tâches spécifiques.
  • Commandes : Disent à l’ordinateur quoi faire.

2.3 Accéder à des documents externes : fichiers CSV et Excel

Souvent, les données sont stockées dans des fichiers externes tels que des fichiers CSV ou des feuilles de calcul Excel. Ces fichiers peuvent être importés dans R pour analyse.

2.3.1 Qu’est-ce qu’un fichier CSV ?

Un CSV (Comma-Separated Values) est un fichier texte où chaque ligne représente une ligne de données, et les colonnes sont séparées par des virgules. Les fichiers CSV sont largement utilisés pour stocker des données tabulaires.
